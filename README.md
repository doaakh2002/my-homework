

## الهدف من البرنامج
برنامج بلغة C يقوم بالآتي:
1. قراءة اسم المستخدم من الإدخال.
2. إلحاق النص "CE-ESY" بنهاية الاسم.
3. تخزين كل حرف من النتيجة في مخزن مؤقت دائري (Circular Buffer).
4. استرجاع الأحرف من المخزن وطباعتها على الشاشة.

## طريقة عمل المخزن المؤقت الدائري
- الحجم الثابت: 15 حرفًا (#define SIZE 15).
- يتم الكتابة والقراءة باستخدام مؤشرات head و tail.
- عند امتلاء المخزن: يظهر تحذير Buffer Overflow.
- عند القراءة من مخزن فارغ: يظهر تحذير Buffer Underflow ويتم إرجاع حرف فارغ.

## الدوال المستخدمة
| الدالة | الوظيفة |
|--------|----------|
| init | تهيئة المخزن (head = tail = count = 0) |
| isFull | التحقق من الامتلاء |
| isEmpty | التحقق من الفراغ |
| write | إضافة حرف إلى المخزن |
| read` | قراءة حرف وإزالته من المخزن |
